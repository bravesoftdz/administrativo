  P R O J E T O

  Este é um projeto que tem por objetivo desenvolver um conjunto de classes para
  a leitura e gravação dos arquivos referentes ao Projeto da Nota Fiscal Eletrônica.



  P R O P O S T A

- Manipular o arquivo XML com sendo um arquivo texto.
- Possilitar a conversão de XML para TXT e de TXT para XML.
- Manter a compatibilidade com as versões antigas do Delphi.
- Liberar os códigos sob os termos da GNU - Lesser General Public License LGPL.
- A linguagem de programação que será utilizada é o Pascal/Delphi.



  D I R E T Ó R I O S

- source            -> classes do projeto, o library path do delphi deve apontar esta pasta.
- source/MunIBGE    -> Arquivos com os nomes dos municípios
